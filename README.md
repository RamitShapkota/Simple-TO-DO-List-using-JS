# Mini Task Manager (Todo App)

A simple Todo application built with **Vanilla JavaScript** to practice core **DOM manipulation** and event handling.

## Features
- Add, complete, and delete tasks
- Dark mode toggle
- Input validation and dynamic UI updates

## Tech Stack
- HTML
- CSS
- JavaScript (DOM)

## Purpose
Created as a learning project to understand how JavaScript interacts with the DOM in real-world web applications.
